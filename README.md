# Polymarket Analytics

Polymarket prediction market analytics - data insights + trading assistance.

## Features

- **Market Scanner**: Real-time price/liquidity ranking across all markets
- **Probability Analysis**: Multi-dimensional team/event probability comparison
- **Historical Data**: Price charts, volume analysis
- **Arbitrage Detection**: Cross-market arbitrage opportunities
- **Alert Notifications**: Price change alerts

## Data Sources

- Polymarket Gamma API (free public)
- Polymarket CLOB API (auth required)

## Install

```bash
pip install -r requirements.txt
python main.py
```

## Strategy Ideas

- Liquidity coverage: high-liquidity markets have lower slippage
- Probability deviation: detect bias vs fundamentals
- Fund flow: track large capital movements
